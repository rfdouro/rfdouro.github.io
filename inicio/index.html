# Introdução

A proposta deste documento é apresentar conceitos de implementação comumente utilizados no estudo de Estruturas de Dados.

As implementações aqui utilizadas serão abordadas com auxílio da linguagem de programação Java.

Este conteúdo leva em consideração os dados apresentados no excelente trabalho do professor João Araújo (opens new window) no conteúdo traduzido e adaptado de Pat Morin disponibilizado em Creative Commons no endereço:

  • http://www.araujo.eng.uerj.br/opendata/ods-ptbr-java.pdf (opens new window).

# Objetivos

  1. Definir princípios básicos de estruturas de dados
  2. Definir e implementar estruturas de dados
  3. Estudar e implementar algoritmos de busca e ordenação de dados
